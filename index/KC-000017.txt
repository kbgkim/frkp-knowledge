Windows 사전 준비 WSL(Windows Subsystem for Linux)을 사용해 Windows 위에서 Ubuntu를 구동합니다. 설치 전 가상화 기능과 Windows 기능이 활성화되어 있어야 합니다. 1.1 가상화 활성화 여부 확인 • 작업 관리자 → [성능] 탭 → CPU 항목의 ‘가상화 : 사용’ 으로 표시되어 있는지 확인 그림 1.1 작업 관리자에서 가상화 활성화 확인 1.2 Windows 기능 켜기 • 제어판 → [프로그램] → [Windows 기능 켜기/끄기] 진입 • ‘Linux용 Windows 하위 시스템’ 과 ‘가상 머신 플랫폼’ 두 항목 체크 • 두 항목을 활성화한 후 PC 재부팅 그림 1.2 Windows 기능 켜기/끄기 화면 1.3 WSL 설치 Windows PowerShell을 관리자 권한으로 실행한 뒤 아래 명령어 입력 wsl --install 그림 1.3 PowerShell에서 wsl --install 실행 1.4 Ubuntu 계정 및 비밀번호 설정 WSL 설치 후 자동으로 Ubuntu가 실행되면 사용자 계정과 비밀번호를 입력해 등록합니다. 그림 1.4 Ubuntu 최초 실행 시 계정/비밀번호 설정 2. Ubuntu 기본 환경 설정 Ubuntu를 실행한 뒤 터미널에서 다음 명령들을 순서대로 입력합니다. 2.1 시스템 업데이트 APT 패키지 인덱스 및 설치된 패키지 업데이트 sudo apt update -y sudo apt upgrade -y 2.2 OpenJDK 8 설치 Java 8 (JDK) 설치 sudo apt install -y openjdk-8-jdk 2.3 R 기본 패키지 설치 R 베이스 패키지 설치 sudo apt install -y r-base 2.4 Linux 개발도구 설치 R · Oracle 라이브러리 빌드에 필요한 시스템 의존 라이브러리 일괄 설치 sudo apt install -y libcurl4-openssl-dev libssl-dev libxml2-dev libfontconfig1-dev libharfbuzz-dev libfribidi-dev libfreetype6-dev libpng-dev libtiff5-dev libjpeg-dev unzip 3. Miniconda 설치 및 가상환경 구성 R 3.5.3 버전을 격리된 환경에서 사용하기 위해 Miniconda를 설치하고 전용 가상환경(r353)을 만듭니다. 3.1 Miniconda 설치 및 실행 설치 스크립트 다운로드 후 실행 wget https://repo.anaconda.com/miniconda/Miniconda3-latest-Linux-x86_64.sh bash Miniconda3-latest-Linux-x86_64.sh [참고] 설치가 끝나면 터미널을 종료한 뒤 다시 실행해야 conda 명령어가 인식됩니다. 3.2 Conda 사용 약관 동의 main / r 채널에 대한 약관을 자동 수락 처리 conda tos accept --override-channels --channel https://repo.anaconda.com/pkgs/main conda tos accept --override-channels --channel https://repo.anaconda.com/pkgs/r 그림 3.1 Conda 약관 동의 화면 3.3 R 3.5.3 가상환경 생성 및 활성화 R 3.5.3 가상환경 생성 conda create -n r353 r-base=3.5.3 -c conda-forge -y 가상환경 활성화 conda activate r353 그림 3.2 가상환경이 켜진 상태 (좌측 프롬프트가 base → r353 으로 변경) 4. Oracle Instant Client 설치 5.1 Oracle Instant Client 다운로드 아래 페이지에서 Linux x86-64 버전을 다운로드합니다. https://www.oracle.com/database/technologies/instant-client/linux-x86-64-downloads.html 그림 5.1 Oracle Instant Client 다운로드 페이지 그림 5.2 다운로드 패키지 선택 다운로드 대상 • Version : 19.30.0.0.0 (Requires glibc 2.14) • Basic Package (ZIP) : instantclient-basic-linux.x64-19.30.0.0.0dbru.zip • SDK Package (ZIP) : instantclient-sdk-linux.x64-19.30.0.0.0dbru.zip 5.2 압축 풀기 예시는 /opt/oracle 에 설치하는 경우입니다. 붉은 글씨 부분을 내 PC의 경우에 맞춰 바꾼 후 실행합니다. cd / opt/oracle unzip instantclient-basic-linux.x64-19.30.0.0.0dbru.zip -d / opt/oracle unzip instantclient-sdk-linux.x64-19.30.0.0.0dbru.zip -d / opt/oracle 그림 5.3 압축 해제 진행 화면 [참고] 덮어씌울지 물어볼 경우 ‘A’ (전부 덮어쓰기) 를 입력합니다. 5.3 환경 변수 설정 ~/.bashrc 에 환경 변수 추가 echo 'export LD_LIBRARY_PATH=/ opt/oracle /instantclient_19_30:$LD_LIBRARY_PATH' >> ~/.bashrc echo 'export PCI_INCLUDES=/ opt/oracle /instantclient_19_30/sdk/include' >> ~/.bashrc source ~/.bashrc [참고] 환경 변수 적용을 위해 터미널을 종료한 뒤 다시 실행하세요. 6.
